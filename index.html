IBM Streams endpoint-monitor

Overview

Endpoint-monitor is an Openshift application template that exposes REST endpoints within an IBM Streams running job as REST endpoints through an Nginx reverse proxy service.

The service supports a single Streams instance with one or more jobs.

The service may be exposed outside of the Openshift cluster using a route.

Endpoint swagger

Generated swagger for the endpoints can be viewed at swagger-ui.html

Streams job endpoints

Python Streams applications

Endpoints are supported by the streamsx.endpoint package.

SPL Streams applications

Endpoints are instances of REST operators from the com.ibm.streamsx.inetserver toolkit.
